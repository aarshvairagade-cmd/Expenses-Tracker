# 💰 Expense Tracker

A simple and responsive Expense Tracker web application built using HTML, CSS, and JavaScript. This app helps users manage their daily income and expenses with real-time balance updates.

---

## 🚀 Features

- ➕ Add income and expense transactions  
- ❌ Delete transactions  
- 💰 Real-time balance calculation  
- 📊 Separate income & expense tracking  
- 💾 Data stored using Local Storage (no data loss on refresh)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
